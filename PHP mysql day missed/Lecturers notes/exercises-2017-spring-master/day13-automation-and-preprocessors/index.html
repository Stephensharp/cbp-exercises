Example with other server.

This one is not standalone, you should copy only package.json and gulpfile.js to your PHP project and run npm install.

You'll probably need to change several paths in gulpfile.js like where to store your css, where to find sass files, what is url of your local server etc.

Also add node_modules to your .gitignore file to prevent accidental commiting of this huge and unnecessary folder.
